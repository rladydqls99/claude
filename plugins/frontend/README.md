# Frontend Plugin

Claude Code용 프론트엔드 개발 도구 및 스킬 플러그인입니다.

## 설치 방법

```bash
# 마켓플레이스 추가
/plugin marketplace add https://github.com/rladydqls99/claude

# 플러그인 설치
/plugin install frontend
```

## 포함된 스킬

### nextjs-fsd-starter

Feature-Sliced Design (FSD) 아키텍처를 적용한 프로덕션 레디 Next.js 프로젝트를 생성합니다.
TypeScript, Tailwind CSS, ESLint, Prettier, Husky 등 모던 프론트엔드 개발 환경이 자동으로 구성됩니다.
